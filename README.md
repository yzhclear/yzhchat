## 项目简介
### 1.
这是一款招聘APP,实现基本的用户登录注册及注销功能,BOSS与应聘者之间可以实时聊天

项目前端采用React + React Router + Redux,UI组件使用的是 Antd Design Mobile, 用Node.js的Express框架编写服务器端逻辑, 使用MongoDB来存储用户数据及聊天信息。用户之间实时聊天功能采用socket.io实现。
